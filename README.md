# INSTRUCTIONS

## Configuration

Create a file *credentials.env* with the following variables:


AWS_ACCESS_KEY_ID=

AWS_SECRET_ACCESS_KEY=

AWS_REGION= 

AWS_DEFAULT_REGION= 


## Run


Please run with the following command


```
make

```

then open the concourse web UI *localhost:8080* and enter:

```
user: test
password: test
```
